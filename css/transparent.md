> transparent

Keep these three options in mind (you want #3):

1) Whole element is transparent:

```css
div {
    visibility: hidden; 
}
```

2) Whole element is somewhat transparent:

```css
div {
    opacity: 0.0 - 1.0;
}
```

3) Just the background of the element is transparent:

```css
div {
    background-color: transparent;
}
```
