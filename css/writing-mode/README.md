# writing-mode 

## 语法

|值|含义|其他|
|:---|:---|:---|
|horizontal-tb|水平方向自上而下的书写方式| Default Value|
|vertical-rl|垂直方向自右而左的书写方式|
|vertical-lr|垂直方向内内容从上到下，水平方向从左到右|
