# 正则表达式

## 正则表达式匹配指定字符区间的内容，且不包含指定字符

|模式|描述|
|:---|:---|
|(?<=%%!)|匹配一个位置,而不是字符.这个位置在 **%%! 后面**|
|(?=%%~)|匹配一个位置,而不是字符.这个位置在 **%%~ 前面**|
|(?<=%%!).*(?=%%~)|%%! 和 %%~ 两个位置之间的所有字符.贪婪匹配模式|
|(?<=%%!).*?(?=%%!)|这两个位置之间的所有字符.懒惰匹配模式，也叫非贪婪模式|


贪婪匹配：匹配到结束<br/>
非贪婪匹配：有多个的会匹配多个

```javascript
const pathname = window.location.pathname // /my/name/is/tom
pathname.match(/(?<=\/).*?(?=\/)/)[0] // my
pathname.match(/(?<=\/).*(?=\/)/)[0] // my/name/is
```
