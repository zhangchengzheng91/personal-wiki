# http VS https

## 什么是 HTTP 协议

HTTP 协议是**超文本传输协议**的缩写，Hyper Text Transfer ProtoCol。他是从 WEB 服务器传输超文本
标记语言（HTML）到本地浏览器的传送协议。

设计 HTTP 最初的目的是为了提供一种发布和接收 HTML 页面的方法。

HTTP 有多个版本，目前广泛使用的是 HTTP/1.1 版本。
